🔧 Connect to Firebase

This is a one-time setup. You only do this once — after this, all workers, check-ins and admins are shared across every phone and device automatically.

1
Go to console.firebase.google.com and sign in with your Google account
2
Click Add project → name it warehouse-checkin → click through and create it
3
In the left menu click Build → Realtime Database → click Create Database → choose any region → select Start in test mode → click Enable
4
Click the ⚙️ gear icon (top left) → Project settings → scroll down to Your apps → click the </> Web button → register the app → copy the firebaseConfig object shown
5
Paste the full firebaseConfig block below and click Connect
